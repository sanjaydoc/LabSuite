Design choice

Status chip — pick a treatment

Exact LabSuite theme (same fonts, same light canvas, same action buttons). All three brighten the red & green and drop the heavy filled blob. Judge each against the outline “mark serviced” / “reorder” buttons beside it, then tell me Option 1, 2, or 3.

Option 1 — Outline pill

bright border + text, transparent · matches the buttons exactly

The chip becomes an outline pill in the same language as the action buttons — only the colour distinguishes status. Cleanest, most monochrome.
